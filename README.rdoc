== README

OURZ is a webapp that tries to reduce the noise of social media platform by consolidating it into a quick feed.

==User Stories
1. A User should be able to follow other users
2. A User should be able to take posts from other social media sites
3. A User should be able to comment and like a post
4. A User post should only last 24hrs
5. A User should be able to find more posts using a Map
